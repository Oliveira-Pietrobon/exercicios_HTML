"Vídeos inspiradores"



video: A tag usada para incorporar vídeos em uma página web. O atributo controls adiciona controles (play, pause, volume, etc.) para o usuário.
source: Define a fonte do vídeo e seu tipo. O atributo src contém o link para o arquivo de vídeo (neste caso, um vídeo de domínio público chamado Big Buck Bunny). O atributo type especifica o formato do vídeo, que é MP4.
width e height: Definem a largura e a altura do player de vídeo na página.
Caso o navegador não suporte a tag video, o texto alternativo "Seu navegador não suporta a tag de vídeo" será exibido.
