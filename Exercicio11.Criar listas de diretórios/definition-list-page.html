"Termos-chave"

Algoritmo → Conjunto de regras ou passos que um computador segue para resolver um problema.
API (Interface de Programação de Aplicação) →Conjunto de funções e regras que permitem que programas diferentes se comuniquem.
Arquitetura Client-Servidor →Modelo de sistema onde um computador (cliente) solicita serviços a outro computador (servidor).
HTML (HyperText Markup Language) → Linguagem usada para criar páginas da web.
Banco de Dados →Local onde as informações são armazenadas e organizadas para fácil acesso.
Big O (Notação Big O) →Forma de medir o desempenho de um algoritmo conforme o número de dados aumenta.
Bubble Sort →Método de ordenação que compara e troca números vizinhos repetidamente até organizar uma lista.
Peer-to-Peer (P2P) →Modelo de rede onde todos os dispositivos se comunicam diretamente, sem um servidor central.
Código-Fonte →Conjunto de instruções escritas por um programador para criar um software.
Computação Distribuída →Modelo onde várias máquinas trabalham juntas para resolver um problema.
Criptografia →Técnica de codificar informações para protegê-las de acessos não autorizados.
Rede de Computadores → Conjunto de dispositivos conectados para compartilhar informações.
Escalabilidade →Capacidade de um sistema crescer e continuar funcionando bem mesmo com mais usuários ou dados.
Estrutura de Dados → Maneira de organizar e armazenar informações para facilitar seu uso.
Firewall → Sistema de segurança que protege redes e computadores contra acessos não autorizados
Framework → Conjunto de ferramentas e códigos prontos que ajudam no desenvolvimento de software
Thread → Pequena parte de um programa que pode rodar ao mesmo tempo que outras.
